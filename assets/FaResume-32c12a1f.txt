سه سال گذشته در شرکت نوبکا به عنوان مهندس نرم‌افزار کار کرده‌ام        [2021 - اکنون]
نوبکا یک هلدینگ است که روی پروژه‌های متنوع الکترونیکی کار می‌کند...

   • سیستم VOIP: این پروژه روی فناوری‌ای کار می‌کند که به شما اجازه می‌دهد تماس‌های صوتی با استفاده از اتصال اینترنت پهن‌باند به جای خط تلفن معمولی (یا آنالوگ) برقرار کنید. این پروژه نیاز به پنل ادمین برای پیکربندی دارد.
    - طراحی رابط کاربری پنل ادمین
    - پیاده‌سازی داشبورد ادمین با استفاده از HTML، CSS و JS

   • مودم LTE: این پروژه ویژگی‌های متنوعی دارد که این محصول را منحصر به فرد می‌کند.
    - پیاده‌سازی داشبورد ریسپانسیو ادمین با استفاده از React، Redux و Axios
    - پیاده‌سازی اجزای مشترک مانند Radio، Multiselect، Input، Menu Button، Loading و غیره
    - مدیریت ورودی‌ها با استفاده از redux-form و hooks
    - پیاده‌سازی هر صفحه از پروژه و زیرصفحات مربوط به آن
    - نوشتن apiCalls برای هر ویژگی در فایل‌های جداگانه برای یافتن و تغییر آسان
    - تعریف reducers جداگانه برای مدیریت بارگذاری و فرآیندهای اصلی که تغییر خواهند کرد
    - استفاده از Git برای کنترل نسخه پروژه
    - کامنت‌گذاری روی تمام اجزا

   • افزونه Chrome Messenger: اپلیکیشن پیام‌رسان با رمزگذاری سرتاسری
    - طراحی UI و UX اپلیکیشن پیام‌رسان
    - پیاده‌سازی فرانت‌اند پیام‌رسان با استفاده از Typescript، React و Manifest V3 کروم اکستنشن
    - کار روی SDK که کلیدها و سایر عملیات کاربران را با استفاده از APIهای بک‌اند اعتبارسنجی می‌کند

   • AR: تحقیق در مورد استفاده از AR در وب
    - امتحان برخی ایده‌های بازاریابی که روی تمام دستگاه‌ها با استفاده از ARJS کار می‌کنند و ویدئو را با کنترل روی هدف خاص نشان می‌دهند
    - استفاده از Vuforia و Unity برای پیاده‌سازی اپلیکیشن اندرویدی چندهدفه روی یک صحنه خاص
    - ساخت اشیای محصول با فرمت .glb

   • FileManager: نوشتن اپلیکیشن کاملاً کاربردی که با پورت سریال رمزگذاری‌شده ارتباط برقرار می‌کند
    - کمک به طراحی دستورات مانند کپی، رمزگذاری، حذف و غیره در توسعه VHDL
    - تجزیه و تحلیل متن خروجی پورت‌های سریال برای نمایش دایرکتوری‌ها و فایل‌ها با استفاده از C#
    - کار با بایت‌ها برای خواندن فایل‌ها (برای ایجاد در ویندوز) و نوشتن فایل‌ها (روی دستگاه) با استفاده از نمونه پورت سریال
    - پیاده‌سازی ویژگی‌های خاص مانند فشرده‌سازی فایل‌ها، سوزاندن کلید و غیره
    - یافتن موارد استفاده از اپلیکیشن و طراحی UX
    - پیاده‌سازی UI کاربرپسند با تمام کلیدهای میانبری که برای تکمیل مدیریت فایل لازم است

   • SPARK: محصولی است که پارکینگ‌های سازمان‌ها یا بخش‌ها را با استفاده از MQTT مدیریت می‌کند
    - دسته‌بندی مجوزهای کاربران و استخراج موارد استفاده هر کاربر
    - استفاده از Microsoft Visio برای طراحی نمودار چرخه عمر اپلیکیشن
    - طراحی UX بر اساس مجوز هر کاربر

   • smart.nobka: فروشگاه هوشمند برای فروش دستگاه‌های IoT نوبکا هلدینگ
    - طراحی UX و UI وب‌سایت
    - پیاده‌سازی وب‌سایت تجارت الکترونیکی ریسپانسیو
    - طراحی محصول با جمع‌آوری مشخصات و طراحی قالب تصاویر جذاب
    - برنامه‌ریزی برندینگ در رسانه‌های اجتماعی و مارکت‌پلیس‌ها
    - برنامه‌ریزی کمپین‌های تبلیغاتی با طراحی بنرهای گیف جذاب و پیاده‌سازی صفحه فرود مبتنی بر اکشن
    - پیاده‌سازی SEO در هر صفحه با استفاده از طرح‌های معتبر محصول گوگل
    - فعال‌سازی تمام ابزارهای SEO گوگل روی وب‌سایت
    - ضبط و ویرایش ویدئوهای جعبه‌گشایی محصولات
    - استفاده از وب AR برای بازاریابی محصولات با تشخیص تصویر هدف روی وب و نمایش ویدئو جعبه‌گشایی در دنیای واقعی

   • titime: سیستم حضور و غیاب با اپلیکیشن چندسکویی که حضور و غیاب کارکنان را با استفاده از موقعیت مکانی لحظه‌ای تأیید می‌کند.
    - طراحی GUI مبتنی بر SEO برای سایت معرفی
    - پیاده‌سازی سایت سفارشی ریسپانسیو با استفاده از وردپرس
    - نوشتن سناریو و ضبط ویدئو آموزشی برای کاربر و پنل ادمین به‌طور جداگانه
    - استخراج ویژگی‌های اپلیکیشن و طراحی بنرهای سایت با وکتورهای SVG شخصی‌سازی‌شده برای هر ویژگی
    - طراحی کاتالوگ، بنر و فلایرهای نمایشگاه APPEX2024 (نمایشگاه تخصصی اپلیکیشن که در تهران برگزار شد)
    - فعال‌سازی تمام ابزارهای SEO گوگل روی وب‌سایت
